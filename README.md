# lost-and-found
It is a web based platform designed to bridge the gap between people who have lost items and those who have found them .This project is dual-sided dashboard for finders to report items and claimants to search for and claim their missing belongings.

## feature
-** user dashboards:** dedicated panel for both finders and claimants to manage listings.
-** Report an found item:** finders can log details about items they have recovered ( e.g.,item type,location found, date, and description).
-**Search and claim:** claimants can filter through found items tolocate what they lost and initiate a claim process.
