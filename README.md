# University Lost & Found Portal

A web-based platform designed to bridge the gap between people who have lost items and those who have found them. This project features a dual-sided dashboard where finders can report items they've found and claimants can search for and claim their missing belongings.

---

##  Project Overview

The **University Lost & Found Portal** is a front-end web application built as part of a Web Design Assignment. It provides a clean, user-friendly interface that helps university students and staff reconnect with their lost belongings quickly and efficiently.

---

## Features

- **Home Page** — Welcomes users with clear call-to-action cards for reporting found items, searching for lost belongings, and contacting a finder.
-  **Login Page** — Allows existing users to securely log in with their email and password.
-  **Sign Up Page** — Enables new users to register with their full name, email, and password.
-  **Dashboard** — A dual-sided interface featuring:
      - Sidebar navigation with options to report lost/found items, search, view reports, and manage profile.
      - **Recently Lost** panel listing items reported missing with their last known locations.
      - **Recently Found** panel listing items that have been turned in with their discovery locations.
      - User profile display with notification badge.
-**Responsive Design** — Built with a mobile-friendly viewport and flexible layouts.
-  **Search Bar** — Expandable search input in the sidebar for quick item lookup.
